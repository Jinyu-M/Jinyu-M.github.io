ROS

rosbag

-查看数据包中的信息
rosbag info ***.bag
-回放数据包中包含的topic内容（要在另一个终端内先启动ROS master，即roscore）
rosbag play ***.bag
-N倍速回放
rosbag play -r N ***.bag
-循环播放
rosbag play ***.bag -l [--loop]
-暂停
rosbag play --pause ***.bag

命令

tmux

-启动新的会话
tmux new -s name
-退出会话
ctrl B + D
-在会话中新建窗口
ctrl B + C
-在会话中切换窗口
ctrl B + id
-查看所有会话
tmux ls
-回到某会话中
tmux a -t name
-重命名会话
tmux rename -t source target
-关闭某会话
tmux kill-session -t name
